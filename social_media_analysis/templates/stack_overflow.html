Q : What is Maturity of Posting a Question in Stack Overflow?

Maturity of Posting a Question in Stack Overflow is the overall percentage level of a user which he gained while for questions on Stack Overflow.
For each question a percentage will be calculated according some reasonable facts of the question.

Q : What are the facts which have considered to calculate maturity of a question?

For each question the percentage of maturity will be calculated according to below qualities of question.
  • Score of the question[Upvotes - Downvotes]
  • Number of Views
  • Amount of tags
  • Whether the question is edited or not?
  • Whether the question is answered or not?
  • Whether the question is closed or not?

Q : How to find User ID of my Stack Overflow account?

It's the number in the middle of your Profile URL,

Eg: http://meta.stackoverflow.com/users/3349551/nodisplayname
For this link the User Id is 3349551
